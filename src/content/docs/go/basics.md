## Commenting in Go

Commenting in Go is two types: Single-line & Multi-line.

```go
// Commenting in Go
package main

import (
  "fmt"
)
// Single-line Comment
func main() {
  fmt.Println("Hello World")
}

/* This is
a
multi-line
comment */
```
